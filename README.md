# Data Pipe (Event Ingestion Service)

A lightweight event ingestion API built with Node.js and Express, containerised with Docker, and deployed to AWS using ECS Fargate with CI/CD via GitHub Actions.

---

## 🚀 Overview

This project demonstrates a modern cloud-native deployment pipeline:

- API built with Node.js / Express
- Containerised using Docker
- CI/CD with GitHub Actions
- Image storage in AWS ECR
- Runtime via AWS ECS (Fargate)
- Traffic routed through an Application Load Balancer
- Logs captured in AWS CloudWatch
